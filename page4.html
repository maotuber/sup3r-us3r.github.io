 About the project.

PortScan.py aims to check open and closed ports quickly, It prompts you to inform a host that can be LOCAL IP EXTERNAL IP and WEBSITE because it converts the host informed and automatically searches its source IP address, it also asks to inform a start port and an end but if you want to check a only door you can too. it was developed in Python 3.5 and the System used was Arch Linux

  • The part of the code responsible for converting from HOST to IP is:
from socket import *

def escanear():
    menu()
    try:
        host = input("\033[31mDigite o host:\033[1;m ")
        print("")
    except:
        escanear()
    try:       
        ip = gethostbyname(host)
        print("\033[31mEndereco IP ▬▶\033[1;m %s \n" %(ip))
    except:
        print("\033[31mHost invalido.\033[1;m")   
        sleep(3)
        escanear()
