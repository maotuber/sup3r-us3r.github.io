    Sobre o projeto.

    O Spy-Quiz tem por finalidade, ajudar os entusiatas e estudante de tecnologia da informação, com desafios para você resolver e passar para o próximo nível e o mais legal é que tudo roda no próprio terminal, ele foi desenvolvido em Python 3.5 e o Sistema utilizado foi o Arch Linux

    • Desenvolvedores Sup3r-Us3r & M4GN4S3C

    Instalação e uso.

    $ sudo pacman -S git
$ git clone https://github.com/Sup3r-Us3r/Spy-Quiz.git
$ cd Spy-Quiz
$ sudo chmod +x spyquiz.py && sudo chmod +x url.py

    Para usar.

    $ ./spyquiz.py

    Requer python3.5
