About the project.

Spy-Quiz is intended to help enthusiasts and students of information technology, with challenges to solve and move to the next level and the cool thing is that everything runs in the terminal itself, it was developed in Python 3.5 and the System used was Arch Linux

  • Creator Sup3r-Us3r

Installation and use.

$ sudo pacman -S git // or sudo apt-get install git
$ git clone https://github.com/Sup3r-Us3r/Spy-Quiz.git
$ cd Spy-Quiz
$ sudo chmod +x spyquiz.py && sudo chmod +x url.py

To use.

$ ./spyquiz.py

Dependency python3.5
