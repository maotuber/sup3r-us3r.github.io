About the project.

The Spy-Quiz is for those who like to break their heads with challenges, it has 30 levels with differentiated difficulties on decryption, coded source reading, image analysis, mathematical calculations, logic and more, it was developed in Python 3.5 and the System used was Arch Linux

  • Creator Sup3r-Us3r

Installation and use.

$ sudo pacman -S git // Or sudo apt-get install git
$ git clone https://github.com/Sup3r-Us3r/Spy-Quiz.git
$ cd Spy-Quiz
$ sudo chmod +x spyquiz.py // Or give permissions using the of spyquiz menu

To use.

$ ./spyquiz.py

Dependency python3.5, curl, lolcat and sl Or you can install using the spyquiz menu
